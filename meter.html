Medidor/Meter

¿Qué es esto?

Un medidor es un elemento que representa visualmente un valor numérico que puede variar entre un determinado rango.

Por ejemplo, se puede usar un medidor para mostrar el tanto porciento de bateria que le queda a un mobil o para indicar el nivel de seguridad de una contraseña.

IMPORTANTE

  • Para definir un medidor se usa la etiqueta <meter>.
  • Existe un rol meter para decir que un elemento hará de medidor.
  • Este elemento no se tiene que usar si no se sabe cual es el valor máximo del rango.

¿Un medidor accesible?

Pues sí, podemos usar un conjunto de atributos para describir los datos que tiene un medidor.

aria-valuemin="NUMERITO"

Indica el valor mínimo del rango.

aria-valuemax="NUMERITO"

Indica el valor máximo del rango.

aria-valuenow="NUMERITO"

Indica el valor actual del rango.

aria-labelledby

Indica qué elemento HTML describe el medidor. Para indicar cual es se especifica su ID.
